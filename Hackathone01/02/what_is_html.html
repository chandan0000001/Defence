HTML : Hyper Text Markup Language

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: This refers to the ability to create links that connect web pages to one another.
  • Markup Language: This means you use "tags" to surround your content, giving that content meaning and structure.

1: The Heading (<h1> to <h6>)

What it is: Headings are tags used to define titles and subtitles on your page. They are crucial for creating a logical hierarchy and outline for your content.


<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

The Purpose: To structure your document in a way that is understandable to both humans and screen readers.

2. The Paragraph (<p>)

The paragraph tag is the most common tag. It's for grouping sentences and blocks of text together.

Tag: <p>

The Purpose: To define a paragraph of text. You should not use multiple line breaks you should use multiple <p> tags.

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam, hic.c.</p>

  • Example:

5. List Elements (<ul>, <ol>, <li>)

What it is: These tags are used to create lists.

The Tags:

  • <ul>: An Unordered List. It creates a bulleted pointed list.
  • <ol>: An Ordered List. It creates a numbered list.
  • <li>: A List Item. Each item in either type of list must be wrapped in an <li> tag.

The Purpose: To group related items together in a list format.

Example (Nesting in action):

The <li> elements are nested as children inside the <ul> parent.

<ul>
  <li>Apex Legend</li>
  <li>Counter Strike</li>
  <li>SnapDragon X Elite</li>
  <li>Over_clock_CPU</li>
</ul>

<ol>
  <li>Call_Of_Duty.</li>
  <li>Intel I 9 HX </li>
  <li> Acer collab with Apple </li>
  <li>Samsung collab with Apple</li>
</ol>

6: The Anchor Tag (<a>)

What it is: The Anchor tag is what makes the web "hypertext." It is used to create a hyperlink.

The Tag: < a >

The Purpose: To make text (or an image) clickable, allowing users to navigate. 

<a href="https://www.coderarmy.in/">Visit Coder Army</a>

7. The Image Tag (<img>)

The Image tag is a 'self-closing tag' used to embed an image onto your page.

<img>

  • src (source): The path or URL to the image file. This is mandatory.
  • alt (alternative text): A description of the image for screen readers for the visually impaired will read this out and for when the image fails to load.
